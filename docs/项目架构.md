# CharacterCard Studio (CCS) Monorepo 架构

> **CharacterCard Studio** 是一个致力于将 SillyTavern 内容开发“工程化”的解决方案。

本仓库 (Monorepo) 包含 CCS 生态的核心工具与示例项目。

## 仓库结构 (Repository Structure)

```text
CharacterCard_Studio_project/
├── CCS_CLI/                   # [CORE] 核心命令行工具 & MCP Server
│   ├── bin/                   # 可执行文件入口
│   ├── src/                   # 源代码 (TypeScript)
│   ├── docs/                  # CLI 详细设计文档
│   │   ├── 统一架构设计.md     # 核心架构与开发模式 (推荐阅读)
│   │   └── 项目结构说明.md     # 代码目录详解
│   └── package.json
│
├── docs/                      # 全局项目文档
│   ├── 项目架构.md             # 本文档
│   └── 酒馆api调研.md          # 早期调研资料
│
└── (Future workspaces...)     # 未来可能包含 GUI 前端、Web 扩展等
```

## 核心组件：CCS CLI

位于 `CCS_CLI/` 目录下的工具是本项目的核心。它集成了：

1.  **Project Manager**: 初始化、克隆、管理 Character 和 Worldbook 项目。
2.  **Sync Engine**: 实现本地文件系统与 SillyTavern 之间的双向同步。
3.  **MCP Server**: 为 Cursor/Claude 等 AI 编辑器提供上下文感知能力。

详细设计请查阅：[CCS 统一架构设计](CCS_CLI/docs/统一架构设计.md)

## 开发指南

### 1. 环境准备
*   Node.js (v18+)
*   SillyTavern instance (运行在 127.0.0.1:8000)

### 2. 快速开始
```bash
# 进入 CLI 目录
cd CCS_CLI

# 安装依赖
npm install

# 构建工具
npm run build

# 连接到全局 (可选)
sudo npm link
```

### 3. 使用
```bash
st --help
```
